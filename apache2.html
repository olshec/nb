Это "жёсткий" перезапуск:

    sudo /etc/init.d/apache2 restart
    Перезапуск, не убивая процесс, а просто применение новой конфигурации:
    
    sudo /etc/init.d/apache2 reload



    Для перезапуска используйте:

    sudo apache2ctl restart
    Подробнее узнать о команде можно вызвав справку:

    man apache2ctl


    sudo service apache2 restart


    So, to start/stop/reload/restart apache on Ubuntu, you can use:

    sudo start apache2
    sudo stop apache2
    sudo reload apache2
    sudo restart apache2

Прежде чем мы сможем настроить Apache для использования PHP 7.3, нам нужно отключить старую версию PHP 7.0, набрав:

a2dismod php7.0
 

Теперь включите только что установленную версию PHP 7.3 с помощью следующей команды:

a2enmod php7.3
 

Перезапустите веб-сервер Apache, чтобы изменения вступили в силу:

    systemctl restart apache2
